Summary: Latitude vs. X

This project analyzes weather pattens relative to the equator using weather data from the OpenWeatherMap API
To perform this analysis we looked at temperature, cloudiness, wind speed, and humidity metrics in juxtaposition to a latitude variable. Charts were plotted using Matplotlib.
